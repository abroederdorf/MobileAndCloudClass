Welcome to this how-to guide for creating an app using Google Maps with Ionic framework 1.0. This guide includes adding custom markers with information windows and getting the location for an address through reverse geocoding. The guide is based on a demo that creates a basic app with side menu navigation for viewing building permits in Seattle using open source data accessed through the SODA consumer API. There is documentation for implementing a basic map with Ionic using a blank template as well as documentation for all of the various components individually. However, there is no documentation that goes through the process of integrating everything to create a map with custom data in a side menu project using the Ionic framework, so this guide serves to expand upon current information to help fill that void.

This how-to guide will step through the process of getting started, first adding a simple map and then adding custom markers and information windows based on data retrieved from an API. This guide presents files in their entirety when possible so that users are able to view the entire context of how the code works together. Line numbers are thus referenced within the guide to explain smaller sections of the code within each of these files.

About Ionic

Ionic is an HTML5 framework that leverages web technologies like HTML5, CSS, and JavaScript to build hybrid apps. Combining this with Cordova, which provides access to the native capabilities of a phone, allows a developer to create a cross-platform app utilizing a single code base.

While there are several options that can be used with Ionic to build an app, this demo will focus on using Cordova and AngularJS to create an Android app. Tools needed include Node.js, Ionic, and Cordova. There will also be a couple of APIs used that will require API keys. The specific commands shown are for developing on Windows and using a command window.
